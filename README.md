# FutureMe — AI симулятор жизненных решений
FutureMe — это iOS-приложение, которое помогает исследовать возможные последствия жизненных решений.  
Пользователь вводит решение, которое он рассматривает, после чего приложение задаёт уточняющие вопросы и с помощью AI моделирует несколько возможных сценариев будущего.
Приложение визуализирует потенциальные исходы в виде ветвящейся временной линии, позволяя увидеть разные пути развития событий и их вероятности.

# Основные возможности
- Ввод решения пользователя (например: переезд, смена работы, запуск проекта)
- Генерация уточняющих вопросов с помощью AI
- Симуляция возможных сценариев будущего
- Отображение альтернативных путей развития событий
- Вероятности различных исходов
- Просмотр деталей каждого сценария
- История предыдущих симуляций

# Используемые технологии
- Swift
- SwiftUI
- MVVM архитектура
- URLSession для сетевого слоя
- OpenAI API для генерации вопросов и сценариев
- Antigravity (для генерации кода) 
- Gemini (для генераций картинок для статей)
